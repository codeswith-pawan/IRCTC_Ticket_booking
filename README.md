# 🚆 IRCTC Ticket Booking System

A Python-based IRCTC-like ticket booking system with features like seat allocation, train selection, availability check, and PDF ticket generation.

---

## ✨ Features

- 🎫 Ticket Booking
- 💺 Seat Allocation (AC & Sleeper)
- 🚆 Train Selection
- 📊 Seat Availability Check
- ⏳ Waiting List System
- 📄 PDF Ticket Generation
- 🔍 Search Ticket by PNR
- ❌ Cancel Ticket

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
